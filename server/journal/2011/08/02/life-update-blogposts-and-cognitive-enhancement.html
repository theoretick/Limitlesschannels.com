Life Update - Blog posts and Cognitive enhancement

02 Aug 2011 | journal | author: theoretick

So I've decided to start getting more serious and frequent on this blog.  I'll be trying a daily blogging experiment today that I'll be extending for the next two months, or as far as I can actually keep this up.  We'll see.

Simultaneously, today is the second day of my two month nootropics/cognitive-training regiment.  I've had an interest in smart drugs (often called nootropics) for quite some time now and while I've played with them in the past, any experiments I've done have lacked in some serious scientific rigor.  So, starting yesterday I've mapped out the next two months with twice daily doses of nootropics, once a day general vitamins, and some nice spreadsheet-directed cataloguing.  I should be posting some kind of updates in the next few weeks but if not I'll definitely have some nice data by the end of my session to post.

Each day I'll be cataloguing sleep, mood, energy, and exercise on a scale of 0 to 5 as well as participating in 20 daily N-Back sessions and using my long-time friend Anki for spaced-repetition memory testing.  N-Back research has pointed to near linear improvement so significant improvement via the noots should increase the slope by N factors, whatever that may be.  Color me hopeful but skeptical.

Daily:

  • Multivitamin

  • B Complex

  • DHA

Twice daily:

  • Piracetam 800mg

  • Choline 500mg

  • Vinpocetine 10mg

  • Huperzine 200mcg

  • DMAE (and later) ALCAR
