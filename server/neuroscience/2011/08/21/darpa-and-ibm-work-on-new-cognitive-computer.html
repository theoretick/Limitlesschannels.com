DARPA and IBM work on new 'Cognitive Computer'

21 Aug 2011 | neuroscience | author: theoretick

Nature has an interview with a fellow involved in the new DARPA/IBM project on building a chip that emulated a cognitive network by emulating neuron interactions between transistors and RAM (as opposed to the traditional chip which uses a CENTRAL processing unit as the central executive.  Think network model and not monolithic architecture.
